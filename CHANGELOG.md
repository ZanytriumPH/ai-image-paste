All notable changes to the "ai-image-paste" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.7] - 2026-03-25

📝 Optimization: Fully refactored the README and package.json descriptions for an improved international reading experience.

## [0.0.6] - 2026-03-25

📝 优化：将说明文档中的“发行说明”迁移至更改日志 (CHANGELOG) 中，保持主页整洁。

📝 Optimization: Migrated the "Release Notes" from the README to the CHANGELOG to keep the main page clean.

## [0.0.5] - 2026-03-25

📝 优化：说明文档对设置快捷键的补充及简洁化。

📝 Optimization: Supplemented and simplified the documentation regarding custom shortcut settings.

## [0.0.4] - 2026-03-25

🚀 打包升级：使用 esbuild 进行代码构建与瘦身，插件体积从 45MB 极速优化至 330KB！

✨ 新增：默认提示词中添加“识别到数学公式则自动转为 LaTeX 格式”的功能。

📝 优化：说明文档添加动态数据徽章 (Badges)。

🚀 Build Upgrade: Switched to esbuild for code building and minification, drastically reducing the extension size from 45MB to 330KB!

✨ Added: Added a rule to the default prompt to automatically convert recognized mathematical formulas into LaTeX format.

📝 Optimization: Added dynamic data badges (Installs, Version, Rating) to the README.

## [0.0.3] - 2026-03-24

🛠️ 优化：解除 API URL 和 Model 的下拉列表限制，全面支持用户手动输入自定义的大模型地址与模型名称。

📝 优化：在设置面板中增加了主流视觉大模型配置的参考说明。

🛠️ Optimization: Removed the dropdown list restrictions for API URL and Model, fully supporting manual input of custom LLM endpoints and model names.

📝 Optimization: Added reference guides for mainstream Vision LLM configurations in the settings panel.

## [0.0.2] - 2026-03-24

🚀 核心架构升级：抛弃传统基础 OCR API，全面接入多模态视觉大语言模型 (Vision LLM)。

✨ 新增：极其精准的 Markdown 表格格式化能力。

✨ 新增：智能识别代码片段，并自动添加 Markdown 行内代码高亮。

🛠️ 优化：支持在设置中自定义系统 Prompt (提示词)，解锁更多 AI 识别玩法。

🚀 Core Architecture Upgrade: Abandoned traditional basic OCR APIs and fully integrated multimodal Vision Large Language Models (Vision LLM).

✨ Added: Extremely accurate Markdown table formatting capabilities.

✨ Added: Smart recognition of code snippets with automatic Markdown inline code highlighting.

🛠️ Optimization: Supported customizing the system prompt in settings, unlocking more AI recognition possibilities.inline code highlighting.

## [0.0.1] - 2026-03-24

初始测试版本发布，基于传统 OCR API 实现基础功能。

Initial test release, implementing basic features based on traditional OCR APIs.
